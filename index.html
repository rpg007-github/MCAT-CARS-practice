🔍 Read for Structure

Find the main argument in the first and last paragraph. Each body paragraph usually supports, qualifies, or contrasts with that central claim. Ask: "What is the author's opinion and how is it organized?"

📝 Author's Tone

Notice evaluative language: "unfortunately," "clearly," "surprisingly," "problematic." These signal the author's attitude. Tone questions are almost always answered by these signal words, not by the topic itself.

⚠️ Extreme Answers

Words like "always," "never," "only," "impossible," "completely" usually make an answer wrong. CARS authors hedge and qualify. Prefer answers with language like "tends to," "often," or "can."

🔃 Strengthen / Weaken

For these questions, identify the specific claim being strengthened or weakened before reading the options. The correct answer directly addresses that claim, not some related topic.

🕐 Passage Timing

Target 8–9 minutes per passage (read + answer all questions). If you spend more than 5 minutes just reading, you're reading too deep. Skim for structure on the first pass, then return for details only as questions demand.

👥 Main Idea Trap

Main idea questions often have a "too broad" and a "too narrow" wrong answer. The correct answer captures the author's central argument without adding claims the author never made. Beware of answers that are factually true but not the author's point.
